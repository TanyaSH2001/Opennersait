Как это работает?

  1. Пользователь делает запрос

    Для этого мы создали
    удобный интерфейс
  2. Программа использует
    специальные библиотеки

    webbrowser-это библиотека , которая
    представляет собой удобный контроллер
    web -browser.
  3. web -browser
    с интернет серверов

    программа запускает нужный контент
  4. Преобразует запрос
    в голос и озвучивает

    Для преобразования текста
    в голос программа использует
    библиотеку SpeakerPy
